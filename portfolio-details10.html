Project information

  • Project Title: Bio Inspired Kangaroo Robot
  • Affiliation: Columbia University
  • Project Duration: Jan 2022 - Present

Project Description

Main Objective of this project is understand and develop all the process involved in developing a novel bi-pedal bio Inspired Kangaroo Robot. This project covers all aspects of robotics from CAD, Mechatronics, 3D Prining, Control and Machine Learning. (This is an Ongoing Project)
